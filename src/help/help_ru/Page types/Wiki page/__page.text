Другим типом страниц, поддерживаемых OutWiker'ом являются страницы в викинотации. На этих страницах по-прежнему можно использовать HTML, но в первую очередь эти страницы рассчитаны на синтаксис, напоминающий (но не точно копирующий) нотацию, которая используется в [[pmWiki -> https://pmwiki.org/]].

Синтаксис викинотации описывается во вложенных заметках.
