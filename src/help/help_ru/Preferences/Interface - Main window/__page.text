Поле ввода '''Формат заголовка главного окна''' позволяет установить, как должен выглядеть заголовок главного окна программы. В этом поле можно использовать макросы подстановки '''{page}''' и '''{file}''', которые будут заменены на заголовок текущей страницы и дерева заметок в целом соответственно.

С помощью переключателя '''Отображать статусную панель''' можно скрывать и показывать панель, расположенную в самом низу главного окна программы.
