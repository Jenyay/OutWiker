Интерфейс - Облако меток




Этот раздел содержит настройки поведения панели "Метки"



С помощью верхних двух параметров можно изменить поведение программы при нажатии левой и средней кнопками мыши на имени метки в данной панели. При нажатии на имя метки может осуществляться поиск страниц, помеченных соответствующей меткой, или осуществляться добавление / удаление метки для выбранной в данный момент страницы.
