HTML-type notes are fully supported by OutWiker. At the moment, the Internet Explorer engine (Trident) is used to render HTML notes on Windows. WebKit engine is used for Linux.

There is one notable difference between standard HTML pages and OutWiker's notes. OutWiker uses the tag <BR> to represent a single line break and <P> to represent a double line break.

There are buttons on the toolbar for adding some of the most frequently used tags.

It is possible to insert links to other pages and attached files.
